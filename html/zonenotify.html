zonenotify

zonenotify può essere utilizzato per inviare un pacchetto di tipo NS_NOTIFY agli slave server BIND. Quando aggiornate i dati di una zona nel DNS che usa djbdns occorre notificare agli slave i cambiamenti avvenuti.
Supporta questo progetto:

Installazione

Scaricate il pacchetto zonenotify-0.1.tar.gz (Perugia, Italy) - zonenotify-0.1.tar.gz (Cleveland, Ohio, USA)

Compilate il programma con: 

    tar -xfz zonenotify-[version].tar.gz
    cd zonenotify
    make
Come utente root installarlo: 
    make install strip

Utilizzo

    zonenotify zone slave1 [slave2] [... slaveN]
così viene inviato un pacchetto "NOTIFY" a slave1 (slave2...slaveN) riguardo la zona "zone", se si riscontano problemi zonenotify visualizzerà l'errore e restituirà 1, altimenti 0.
